# แบบจำลองแก๊สด้วยทฤษฎีจลน์ของแก๊ส

เว็บแอปนี้เป็นโปรแกรมจำลองอนุภาคแก๊สในภาชนะ 2 มิติแบบโต้ตอบได้ เหมาะสำหรับโครงงานวิทยาศาสตร์ระดับนักเรียน/นักศึกษา เพื่อสังเกตความสัมพันธ์ระหว่างอุณหภูมิ ความเร็ว พลังงานจลน์ ความดัน ปริมาตร และจำนวนอนุภาค

## แผนโครงงาน

| ส่วนของโครงงาน | แนวทางที่ใช้ |
|---|---|
| แบบจำลอง | อนุภาคแก๊สเป็นวงกลมเล็กในกล่อง 2 มิติ |
| การเคลื่อนที่ | สุ่มความเร็วเริ่มต้นตามแนวคิด Maxwell-Boltzmann แล้วเคลื่อนที่ต่อเนื่อง |
| การชนผนัง | ชนแบบยืดหยุ่น โดยกลับเครื่องหมายความเร็วตั้งฉากกับผนัง |
| การชนอนุภาค | เปิด/ปิดได้ ใช้การชนแบบยืดหยุ่นของอนุภาคมวลเท่ากัน |
| ค่าที่แสดง | ความเร็วเฉลี่ย พลังงานจลน์เฉลี่ย ความดันเฉลี่ย อุณหภูมิจำลอง พื้นที่ภาชนะ และ PV/N |
| กราฟ | ฮิสโตแกรมการกระจายความเร็ว พร้อมเส้นเปรียบเทียบ Maxwell-Boltzmann แบบ 2 มิติ |
| การทดลอง | มี preset สำหรับเพิ่มอุณหภูมิ ลดปริมาตร และเพิ่มจำนวนอนุภาค |

## หลักการคำนวณในโปรแกรม

โปรแกรมใช้หน่วยจำลอง โดยกำหนด `kB = 1` และใช้ตัวแปร `SPEED_TO_PIXELS` เพื่อแปลงความเร็วเชิงฟิสิกส์เป็นการเคลื่อนที่บน canvas

- ความเร็วเริ่มต้นสุ่มจากการกระจายแบบ Gaussian ของแกน x และ y
- อุณหภูมิในแบบจำลอง 2 มิติสัมพันธ์กับพลังงานจลน์เฉลี่ย:

```text
K_avg = (1/2) m <v^2> = kB T
```

- การชนผนังแบบยืดหยุ่น:

```text
vx -> -vx   เมื่อชนผนังซ้าย/ขวา
vy -> -vy   เมื่อชนผนังบน/ล่าง
```

- อิมพัลส์จากการชนผนัง:

```text
J = 2 m |v_perpendicular|
```

- ความดันเฉลี่ยประมาณจากอิมพัลส์ต่อเวลาและต่อความยาวผนัง
- ในแบบจำลอง 2 มิติ ปริมาตรแทนด้วยพื้นที่ของกล่อง จึงใช้ความสัมพันธ์เชิงเปรียบเทียบ:

```text
P A ≈ N kB T
```

ดังนั้นค่า `PV/N` บนหน้าจอควรอยู่ใกล้กับอุณหภูมิจำลองโดยประมาณ แต่จะมีการแกว่งเพราะจำนวนอนุภาคจำกัดและการชนเป็นแบบสุ่ม

## ตัวอย่างการทดลอง

1. เพิ่มอุณหภูมิ: กด `อุณหภูมิสูง` แล้วสังเกตความเร็วเฉลี่ย พลังงานจลน์เฉลี่ย ความดัน และฮิสโตแกรมที่เลื่อนไปทางความเร็วสูง
2. ลดปริมาตร: กด `ปริมาตรต่ำ` แล้วสังเกตว่าความดันเพิ่มขึ้นเพราะอนุภาคชนผนังถี่ขึ้น
3. เพิ่มจำนวนอนุภาค: กด `อนุภาคมาก` แล้วสังเกตว่าความดันเพิ่มขึ้นเมื่อจำนวนอนุภาคมากขึ้น
4. สำรวจมวล: เพิ่มมวลอนุภาค โดยคงอุณหภูมิไว้ แล้วดูว่าความเร็วเฉลี่ยลดลงแต่พลังงานจลน์เฉลี่ยยังสัมพันธ์กับอุณหภูมิ

## ข้อจำกัด

แบบจำลองนี้เป็นแบบเรียนรู้ ไม่ใช่แบบจำลองวิจัยจริง ข้อจำกัดสำคัญคือเป็น 2 มิติ ใช้หน่วยจำลอง อนุภาคเป็น hard disk ไม่มีแรงระหว่างโมเลกุล ไม่มีการหมุน/สั่นของโมเลกุล และความดันจากการชนมีความผันผวนตามจำนวนอนุภาคและช่วงเวลาเฉลี่ย
